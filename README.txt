nvid - Compressed video player for the TI-Nspire

Nvid is a compressed video player for the TI-Nspire. It plays vp8-encoded video.
To use:
- Copy nvid.tns to the calculator and run it once to install the file association
- Open ivf files to play them
To convert videos:
- Drag files over "drop files here.bat" in the converter folder. This will create a file with the extension ivf.tns that can be transferred to the calculator
